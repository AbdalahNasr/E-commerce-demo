# E-commerce-demo

## Vercel Deploy

- **Run locally:** `npm install` && `npm start`
- **Build:** `npm run build`
- **Deploy to Vercel:** Import the repository in the Vercel dashboard or run the `vercel` CLI. Vercel will run `npm run build` and deploy the `build` directory. Ensure environment variables (if any) are configured in the Vercel project settings.
